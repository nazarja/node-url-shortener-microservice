Url Shortener Microservice

A freeCodeCamp Javascript Back-End Challange

This challange makes use of NodeJS, ExpressJS, MongoDB and the url-valid npm module.

Challange:

  • To create a URL Shortner Microservice. Given a URL as a parameter, a JSON object will be returned with the shortened url link.
  • Clicking on the newly created shortened url will redirect you to the original link that was input.
  • If an Invalid url is entered, the JSON response will contain an error instead.

Usage:

In the url bar, after the current url. Type /new/ followed by the new url that you wish to shorten.
For example to shorten google.com type /new/http://www.google.com to receive your new shortened url link.
The url you enter must be a valid url or an error will occur.

Example Usage:

Click me: http://localhost:3000/new/https://www.google.com
Click me: http://localhost:3000/new/https://www.freecodecamp.org
Click me: http://localhost:3000/new/https://www.facebook.com
Click me: http://localhost:3000/new/http://www.seanmurphy.eu

Example Output:

Success

{ "Original URL":"https://www.google.com", "Shortened URL":"http://localhost:3000/3Ye" }

Error

{ "Error": "Please make sure you have entered a valid URL.", "Try Again": "http://localhost:3000" }
